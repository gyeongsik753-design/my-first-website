## 컬러 사용 가이드

### 1. Primary Black (`#000000`) — 사용 비율 약 45%

| 적용 대상 | 설명 |
|-----------|------|
| 히어로 섹션 배경 | 풀 와이드 블랙 배경으로 강렬한 첫인상 |
| 섹션 배경 (교차) | 화이트 섹션과 번갈아가며 리듬감 형성 |
| 제목 텍스트 | 라이트 배경 위에서 강한 가독성 |
| 버튼 기본 상태 | CTA 버튼의 기본 배경색 |
| 푸터 배경 | 페이지 마무리 영역 |

```jsx
/* MUI 적용 예시 */
<Box sx={{ bgcolor: '#000000', color: '#FFFFFF', p: 4 }}>
  <Typography variant="h2">히어로 타이틀</Typography>
</Box>
```

### 2. Secondary White (`#FFFFFF`) — 사용 비율 약 38%

| 적용 대상 | 설명 |
|-----------|------|
| 콘텐츠 섹션 배경 | About Us, 상품 그리드 등 정보 전달 영역 |
| 다크 배경 위 텍스트 | 블랙 배경에서의 모든 텍스트 |
| 버튼 텍스트 | 블랙 버튼 위 라벨 |
| 카드 배경 | 상품 카드 기본 배경 |

```jsx
<Box sx={{ bgcolor: '#FFFFFF', color: '#000000', p: 4 }}>
  <Typography variant="body1">콘텐츠 영역</Typography>
</Box>
```

### 3. Gray Spectrum (`#333333` ~ `#999999`) — 사용 비율 약 10%

| 색상 | 용도 |
|------|------|
| `#333333` | 보조 텍스트, 버튼 호버, 셰브론 어두운 톤 |
| `#555555` | 셰브론 중간 톤, 아이콘 |
| `#666666` | 설명 텍스트, 캡션, 날짜 표시 |
| `#999999` | 비활성 텍스트, 셰브론 밝은 톤 |
| `#E0E0E0` | 카드 테두리, 구분선 |

```jsx
<Typography sx={{ color: '#666666', fontSize: '0.875rem' }}>
  부제목 또는 설명 텍스트
</Typography>
```

### 4. Accent Red (`#C8102E`) — 사용 비율 약 5%

| 적용 대상 | 설명 |
|-----------|------|
| 브랜드 로고 포인트 | "G" 로고의 레드 컬러 |
| 링크 호버 상태 | 마우스 오버 시 시선 유도 |
| 배지/태그 | "NEW", "HOT" 등 강조 라벨 |
| 에러 상태 | 폼 유효성 검사 에러 표시 |

```jsx
<Button
  sx={{
    bgcolor: '#C8102E',
    color: '#FFFFFF',
    '&:hover': { bgcolor: '#A00D24' }
  }}
>
  특별 강조 버튼
</Button>
```

## 반응형 고려사항

### 다크모드 대응

이 디자인은 이미 블랙 기반이므로 라이트/다크 모드 전환 시 역전 방식을 사용합니다.

```css
/* 라이트 모드 (기본) */
:root {
  --color-bg-primary: #000000;
  --color-bg-secondary: #FFFFFF;
  --color-text-primary: #000000;
  --color-text-on-dark: #FFFFFF;
}

/* 다크 모드 — 화이트 섹션만 다크로 전환 */
@media (prefers-color-scheme: dark) {
  :root {
    --color-bg-secondary: #121212;
    --color-bg-subtle: #1E1E1E;
    --color-text-primary: #E0E0E0;
    --color-text-secondary: #AAAAAA;
    --color-text-muted: #888888;
    --color-border-light: #333333;
    --color-accent: #FF2D4F;         /* 다크 배경에서 더 밝은 레드 */
  }
}
```

### 접근성 (Accessibility) 체크리스트

| 조합 | 명도비 | WCAG 등급 |
|------|--------|-----------|
| `#000000` on `#FFFFFF` | 21:1 | AAA |
| `#FFFFFF` on `#000000` | 21:1 | AAA |
| `#666666` on `#FFFFFF` | 5.74:1 | AA |
| `#C8102E` on `#FFFFFF` | 5.56:1 | AA |
| `#C8102E` on `#000000` | 3.78:1 | AA Large |
| `#AAAAAA` on `#000000` | 8.59:1 | AAA |

> 블랙/화이트 기반이므로 대부분의 텍스트 조합이 WCAG AA 이상을 충족합니다.
> `#C8102E`(레드)는 작은 텍스트보다 큰 제목이나 아이콘에 사용하는 것을 권장합니다.
